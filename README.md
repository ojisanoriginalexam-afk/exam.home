# おじさんの試験対策ページ — 店舗（フロント）リポジトリ

このリポジトリは「おじさんの試験対策ページ」の **店舗ページ（フロント）** です。
GitHub Pages で公開され、利用者がアクセスする入口になります。

- 公開URL: `https://ojisanoriginalexam-afk.github.io/exam.home/`
- 中身
  - `index.html` … 店舗ページ本体（商品一覧・ログイン／新規登録・購入・ライブラリ）
  - `tokushoho.html` … 特定商取引法に基づく表記
  - `study-hero.jpg` … トップ画像

問題（テスト）の実体は、別リポジトリ **me2-exam** にランダム名フォルダで置かれています。

## 連携しているもの

- バックエンド: Google Apps Script ＋ Google スプレッドシート（ログイン・購入確認・自動同期・閲覧URL配信）
- 決済: Stripe（決済リンク）

運用・更新・トラブル対応のくわしい手順は **「管理者マニュアル.docx」** を参照してください。

## 重要な注意

- このリポジトリは **非公開（Private）** にしてください。公開すると店舗のソースが誰でも読めてしまいます。
- ログイン後・購入後に利用者が戻ってくる先（GAS の `SITE_URL`）は、必ずこのリポジトリのURL（`.../exam.home/`）を指している必要があります。ここが `me2-exam` を指していると、トップページが無いため README が表示されてしまいます。
- `Code.gs`（バックエンド）の中身を変更したときは、GAS を **新バージョンで再デプロイ** する必要があります。ページ（このリポジトリ）の変更は、GitHub に上げ直すだけで反映されます。
